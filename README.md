# 📝 To-Do List em PHP

Projeto de estudo para praticar lógica de programação, PHP orientado a objetos e manipulação de banco de dados com PDO.

## Funcionalidades
- Criar tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Rolagem Vertical

## Tecnologias utilizadas
- PHP (PDO)
- MySQL
- HTML/CSS
- Font Awesome

## Como executar
1. Clone o repositório
2. Importe o arquivo `db.sql` no seu MySQL
3. Configure as credenciais do banco em `conexao.php`
4. Rode em um servidor local (XAMPP, Laragon, etc)
